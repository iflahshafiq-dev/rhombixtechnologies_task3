##  MyBookShelf — Personal Library Manager

A clean and responsive web app to manage your personal book collection.

### Features
- Dashboard with live reading stats and genre breakdown
- Add, edit, and delete books with ratings and notes
- Filter and sort library by category, status, or author
- Track borrowed books with due dates and overdue alerts
- Full borrowing history with search
- Data persists via localStorage — no backend needed

### Tech Stack
HTML5 · CSS3 · Vanilla JavaScript · localStorage API

### How to Run
Clone the repo and open `index.html` in any browser. No installation required.
